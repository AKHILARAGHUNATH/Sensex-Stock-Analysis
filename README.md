# 📊 Sensex Stock Market Analysis

## 🔍 Project Overview

This project demonstrates an end-to-end stock market analysis of Sensex data using Python, SQL, and Power BI. It includes data collection, preprocessing, exploratory analysis, and interactive dashboard visualization.

## 🛠 Tools & Technologies

* Python (Pandas, Matplotlib)
* SQL
* Power BI

## 📈 Key Insights

* Identified long-term growth trends in Sensex
* Analyzed impact of major events like COVID-19, Demonetization, and GST
* Performed monthly and yearly trend analysis
* Observed volatility patterns in the stock market

## 📂 Project Structure

* **Dashboard/** → Power BI dashboards
* **Data/** → Sample dataset
* **Scripts/** → Data collection script
* **Analysis/** → Jupyter notebooks & graphs

## 🚀 How to Run

1. Run `data_collection.py` to fetch data
2. Use dataset from Data folder
3. Open `.pbix` files in Power BI

## 💡 Conclusion

This project highlights strong skills in data analysis, visualization, and understanding of financial market trends.
